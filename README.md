# Sesugh Barnabas — Data Science Portfolio

Personal portfolio website for Sesugh Barnabas, a final-year Data Science student at the University of Sunderland.

**Live site:** https://sesugh-dev01.github.io

## About

A responsive, single-page portfolio showcasing projects, technical skills and contact details. Built for graduate job applications in Data Science, Business Intelligence and Analytics across the UK.

## Sections

| Section | Description |
|---------|-------------|
| About | Bio, stat strip and availability for graduate roles |
| Projects | Six projects across Python, Data Analysis, AI/ML and Web Dev |
| Skills | Animated skill bars and tech tag groups (BI, Databases, Dev Tools) |
| Contact | Form with validation and direct contact details |

## Tech Stack

- HTML5 / CSS3 / JavaScript (vanilla, no frameworks)
- Font Awesome icons
- jQuery + Poptrox (lightbox)
- Hosted on GitHub Pages

## Projects Featured

1. **Automated Data Cleaning Pipeline** — Python, Pandas, NumPy
2. **UK Crime Trends Analysis** — Python, SQL, Matplotlib
3. **Student Grade Predictor** — scikit-learn, Random Forest (84% accuracy)
4. **COVID-19 Data Dashboard** — JavaScript, REST API
5. **Retail Sales ETL Pipeline** — Python, SQL, SQLite
6. **Portfolio Website** — HTML, CSS, JavaScript, GitHub Pages

## Running Locally

No build step required. Open `index.html` directly in a browser or serve with any static file server:

```bash
# Python
python -m http.server 8000

# Node (npx)
npx serve .
```
